# Campus SOP Agent

校园小程序需求拆解与 SOP 生成器。项目面向服务产品实习生、AI Agent、Prompt 工程和文档沉淀场景，用来展示“把模糊需求拆成清晰步骤”的能力。

## 项目定位

用户输入一个校园小程序需求后，系统自动生成：

- 功能模块拆解表
- 开发 SOP
- 测试清单
- Prompt 模板
- 可写入简历的项目描述
- Markdown 分析报告

## 适配岗位能力

- 需求分析：将模糊需求拆解为用户角色、页面、字段、依赖和测试点
- SOP 编写：沉淀可重复执行的项目流程
- Prompt 设计：将人工分析流程转化为大模型可理解的任务指令
- AI Agent 思维：按步骤拆解任务，形成可执行链路
- 文档沉淀：支持导出 Markdown 报告
